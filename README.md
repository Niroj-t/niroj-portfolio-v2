# Niroj Portfolio (v2)

A personal portfolio website built with **Next.js (App Router)**, **TypeScript**, and a modern component system (Radix UI + Tailwind CSS) to showcase projects, experience, education, and contact details.

---

## 🚀 Features

- ✅ **Fast, SEO-friendly static pages** (Next.js App Router)
- ✅ **Responsive layout** with mobile-first design
- ✅ **Animated skeleton loaders** for a polished UX
- ✅ **Contact form** with server-side handling & email sending
- ✅ **Modular UI components** (Reusable design system in `components/ui/`)
- ✅ **Dark mode / theme toggle** support

---

## 🧱 Tech Stack

- **Next.js** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **Radix UI** (via custom UI components in `components/ui/`)
- **React** + **Server Components**
- **Email Sending** (mails/send-mail.ts)
